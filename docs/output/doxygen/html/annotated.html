這是附帶簡略說明的類別，結構，聯合型態(unions)及介面(interfaces):
 Nhahahalib
 Nui
 Chahaha_form_about使用自訂標題列基底表單的 About 視窗。 使用者點擊基底區域時只會隱藏，不會直接釋放。
 Chahaha_form_base_titlebar以自訂標題列取代系統標題列的 WinForms 基底視窗。 透過直接處理 Win32 非用戶區訊息，保留拖曳與縮放行為。
 Chahaha_form_cover簡單的封面 / Splash 視窗佔位類別。
 Chahaha_form_version使用共用自訂標題列基底表單的版本資訊視窗。 點擊基底區域時只會隱藏，不會直接釋放。
 CSK_Control專案內給 SkiaSharp 桌面控制項使用的擴充型別。 目前行為等同 SKControl，但保留穩定型別供未來擴充與設計工具使用。
 CTab_Controll_Base可隱藏原生頁籤標頭，但保留分頁切換能力的 TabControl。
 Chahaha_bitmap_reference以手動參考計數共享 EmguCV 的 Mat 與對應的 Bitmap 視圖。 Bitmap 直接指向 Mat 的記憶體，因此兩者生命週期必須一致。
 Chahaha_gps_gnssGNSS / NMEA 解析工具。 目前重點是從 RMC 訊息中擷取經緯度，並盡量避免在高頻解析時產生額外配置。
 Chahaha_json簡單的 JSON 載入/儲存工具類別，適合一般 POCO 物件持久化。 為了維持本函式庫一致風格，這裡使用狀態碼而不是直接拋出例外。
 Chahaha_log
 Chahaha_sleep用於次毫秒等待的混合式休眠工具。 以增加少量 CPU 使用率換取比一般 Thread.Sleep 更好的時間精度。
 Chahaha_thread_command以獨立背景執行緒處理命令佇列的工作類別。 子類別通常覆寫 Handle(hahaha_thread_command_command) 來實作實際命令處理。
 Chahaha_thread_command_command給 hahaha_thread_command 使用的簡單命令封裝物件。 Code_ 表示命令代碼，Parameter_ 攜帶額外參數。
 Chahaha_thread_pause可重複使用的單工作執行緒包裝類別。 每次呼叫 Enabled 都會喚醒執行緒一次，並執行 Handle。
 Chahaha_thread_pool_queue_timerWindows 計時器佇列 API 的薄封裝。 子類別通常覆寫 Handle 來接收計時器觸發事件。
 CHahaha_Timer以 QueryPerformanceCounter 包裝的高精度計時器。 適合量測短時間程式區段的執行時間。
