hahahalib.hahaha_bitmap_reference 類別 參考文件

以手動參考計數共享 EmguCV 的 Mat 與對應的 Bitmap 視圖。 Bitmap 直接指向 Mat 的記憶體，因此兩者生命週期必須一致。 更多...
