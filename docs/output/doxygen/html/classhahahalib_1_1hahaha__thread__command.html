hahahalib.hahaha_thread_command 類別 參考文件

以獨立背景執行緒處理命令佇列的工作類別。 子類別通常覆寫 Handle(hahaha_thread_command_command) 來實作實際命令處理。 更多...

virtual int Create ()

virtual int On_Command ()
 保留給子類別自訂命令建立流程的擴充點。 預設實作不做任何事。
virtual void OnHandleException (hahaha_thread_command_command cmd, Exception ex)
 提供子類別集中處理 Handle 例外的擴充點。
virtual int Reset ()
 將物件重設為關閉狀態，並清空尚未處理的命令佇列。
virtual int Terminate ()
 安全終止輔助方法，只通知結束並等待，不做強制中止。
void Thread_Proc ()
 執行緒主迴圈。 每次被喚醒後會盡量把目前佇列清空，再通知 Wait 可返回。
virtual int Wait ()
 等待目前批次命令處理完成，或等待工作執行緒結束。
